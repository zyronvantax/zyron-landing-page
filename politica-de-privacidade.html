Privacidade e transparência

Política de Privacidade

Atualizada em julho de 2026.

Esta política explica de forma objetiva como a VANTAX utiliza as informações preenchidas no formulário de contato do site.

Dados utilizados no atendimento

O formulário solicita apenas as seguintes informações:

  • nome;
  • número de WhatsApp;
  • empresa ou segmento;
  • tipo de projeto;
  • mensagem enviada pelo usuário.

Finalidade

Os dados são usados para organizar a solicitação, iniciar o atendimento, compreender a necessidade apresentada e conversar sobre um possível projeto. A VANTAX não vende essas informações.

Envio pelo WhatsApp

O formulário não envia os dados automaticamente para um banco de dados da VANTAX. Ao selecionar o botão de envio, o navegador organiza o texto e abre o WhatsApp para que o próprio usuário revise e envie a mensagem. A partir desse momento, o tratamento também está sujeito às práticas e configurações do WhatsApp.

Retenção

As mensagens de atendimento podem ser mantidas pelo período necessário para responder à solicitação, acompanhar a conversa e registrar decisões de um eventual projeto. O usuário pode solicitar a exclusão quando aplicável.

Direitos do usuário

Você pode pedir informações, correção ou exclusão dos dados enviados no atendimento. Para isso, escreva para vantaxcompany@gmail.com, informando dados suficientes para localizar a conversa.

Contato

Dúvidas sobre esta política podem ser enviadas para vantaxcompany@gmail.com. A VANTAX não informa endereço, CNPJ ou encarregado de dados nesta página porque esses dados oficiais não foram fornecidos.
